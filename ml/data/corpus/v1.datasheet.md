# FluenceBench-FR — corpus v1 (tranche teacher, #18)

A teacher-LLM-generated French dialogue corpus, the first scaling beyond the
hand-authored seed v0, used to close the Phase 4 value criterion (#31) with an
**out-of-domain** measurement (ADR-0008).

## Provenance

- **Generator**: `fluence_data.generate` (pure logic in `fluence_data.teacher`).
- **Teacher model**: Gemma 4 E4B instruct (`gemma-4-E4B-it-Q4_K_M.gguf`) served
  by `llama-server`, chat template applied, thinking disabled.
- **Embedder (dedup)**: bge-m3 (`bge-m3-Q8_0.gguf`), cosine near-duplicate
  removal at threshold 0.92.
- **Parameters**: `--per-cell 3 --temperature 0.85 --seed 20260613`, variant
  RNG seed 20260613.
- **Matrix**: 12 situations × 4 registers (SPEC §5.D), 144 drafts → **136**
  dialogues after dedup (8 near-duplicates removed).
- **Splits (frozen, in the data)**: train 98 / dev 19 / test 19; every situation
  is represented in the held-out test split so the value gate measures
  out-of-domain (ADR-0008).

The corpus is **100 % synthetic** (D-8.2): no personal data, not P0. It may be
logged, published and stored in the clear. Intended license for publication:
**CC BY-SA 4.0** (the FluenceBench-FR benchmark).

## Anti-pathos (SPEC §5.D étage 3)

The teacher runs under the anti-pathos consigne (ordinary life — humour, desire,
irritation, plain requests; never misérabilisme / inspiration porn). Screening:

1. **Automatic** flagger (`fluence_data.antipathos`): **0 markers** across the
   136 dialogues.
2. **Human review** of the ≥ 10 % sample (`v1.review.txt`): performed and
   **approved** — framing is ordinary; mentions of fatigue or pain are
   ordinary-life difficulty, not framing of disability.

## Value measurement (Phase 4 T6, #31 / ADR-0008)

Measured locally on the held-out **test** split (n-gram trained on train+dev),
rephrase via the real hub `/suggest`, semantic acceptance by bge-m3 cosine:

| mode | KS% | WPM | acceptance |
|---|---|---|---|
| letter-by-letter | 0.00 | 15.00 | 0.00 |
| n-gram (out-of-domain) | 11.71 | 12.31 | 0.16 |
| **rephrase** | **29.52** | **20.91** | **0.95** |

Gate (ADR-0008): rephrase beats the n-gram on **WPM** (+8.60, primary) **and**
on out-of-domain **KS%** (+17.81) → **PASS**.

## Regeneration

```bash
python -m fluence_data.generate \
  --teacher-url http://127.0.0.1:8080 --embed-url http://127.0.0.1:8090 \
  --out ml/data/corpus/v1.jsonl --per-cell 3 --temperature 0.85
```

The teacher's text is non-deterministic across runs/models; the committed
`v1.jsonl` is therefore the **frozen artifact** (the truth), not a reproducible
build. Full corpus v1 (≥ 500 dialogues per teacher, D-12.2) remains debt (#18).
